## Max Editor for VS Code — first release

One-click editor maximize: hide all VS Code panels and bring them back
exactly as they were.

### Features

- **Toggle maximize** via the editor title bar icon, status bar button, or
  `Cmd+Shift+M` / `Ctrl+Shift+M`
- Hides sidebar, bottom panel, secondary sidebar, activity bar, and status bar
  in one shot
- **Restore** brings everything back — the status bar button changes icon so
  you always know the current state

### Requirements

- VS Code **1.104** or newer

### Install

Download `maxeditor-0.1.0.vsix` from the Assets section below, then:

```bash
code --install-extension maxeditor-0.1.0.vsix
```

Or in VS Code: **Extensions** → `…` menu → **Install from VSIX…**
